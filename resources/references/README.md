# 単元指導計画リファレンス (Recommended References)

ここに、より良い単元指導計画を作成するためのリファレンス資料を格納してください。  
また、以下のリンク先にある「優秀な事例」を参照し、ご自身の学校の指導計画に取り入れることを推奨します。

## 1. 東京都教師道場
- **特徴:** 「指導と評価の一体化」「主体的・対話的で深い学び」の具現化に重点。
- **参考リンク:**
  - [東京都教職員研修センター 学習指導案書式例](https://www.kyo-kai.ed.jp/shoshiki/index.html)
  - [授業力向上に向けた取組](https://www.kyo-kai.ed.jp/kensyu/index.html)

## 2. 国立教育政策研究所 (NIER)
- **特徴:** 「評価規準」の作成における最も標準的な指針。
- **参考リンク:**
  - [「指導と評価の一体化」のための学習評価に関する参考資料](https://www.nier.go.jp/kaihatsu/shidou_hyouka/)

## 3. 福岡県教育委員会
- **特徴:** 小中連携や特別支援教育の視点を取り入れた実践事例が豊富。
- **参考リンク:**
  - [福岡県教育センター 指導案データベース](http://www.educ.pref.fukuoka.jp/)

## 使い方
1. 上記サイトから参考になる学習指導案PDFなどをダウンロードし、このフォルダに保存してください。
2. アプリの生成ボタンを押す前に、これらの資料の内容（特に校内研究テーマに関連する部分）を確認し、生成指示の「ローカルカスタマイズ」欄に入力すると、より精度の高い指導案が作成できます。
